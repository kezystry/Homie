# Glossary

- **Spine** — the five-part loop Perceive → Remember → Reason → Act → Interface. → [[Spine]]
- **Tile** — a self-contained capability (a "living cell") that plugs into the spine. → [[Core and Tiles]]
- **Core** — the minimal substrate: the spine + Security; never imports a tile.
- **Pattern of life** — the household's learned routines, held by [[Remember]].
- **Friction** — corrections (reversal/repeat/remark) the system learns from; silence = approval.
- **Actuator** — a thing Homie can drive (a light), mapped to a home entity by [[Act]].
- **Expectation** — Remember's answer to "what's normal here/now": a rate + a `novel` flag.
- **Tool catalog** — tile functions exposed to the LLM as callable tools. → [[Reason]]
- **Confirmation gate** — `ctx.confirm`; a yes/no answered by gesture or voice. → [[Consent and Gestures]]
- **Mesh / Link** — encrypted node-to-node transport; the bus made node-transparent. → [[Mesh]]
- **Abliterated model** — an uncensored local LLM; safe here because safety is structural.
- **Anchor / cortex** — the always-on box vs the on-demand GPU. → [[Always-on topology]]
